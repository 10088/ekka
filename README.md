
# Ekka

A New Distribution Layer for EMQ R3

```
----------             ----------
| EMQ R3 |<--- MQTT--->| EMQ R3 |
|--------|             |--------|
|  Ekka  |<----RPC---->|  Ekka  |
|--------|             |--------|
| Mnesia |<--Cluster-->| Mnesia |
|--------|             |--------|
| Kernel |<----TCP---->| Kernel |
----------             ----------
```
